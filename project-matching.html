Project Matching

Making onboarding and project management easier across organizations.

Project Date May, 2020
Role Founder, Backend Developer, Core Maintainer
Tech Stack Beta: Node.js, React.js, Firebase
Final: Ruby on Rails, Docker
Visit Beta
GitHub
GitHub (Beta)

Having worked on several projects synchronously, I realized that the greatest challenge faced by tech leads was the inability to find the right people with the needed skills, and for such people to find the right projects.

Thus, my friends Aryaman, Duy and I set out to create a platform that would facilitate this matching.

We created the beta version of project-matching: Projectory. This served as a basic proof of concept which was well recieved by SIG Leaders of the UMass ACM, Google DSCs across the globe and RezScore.

Following the success of the beta, we opened up development for the final product to the open sourced community, under the umbrella of the UMass Google DSC.

I'm currently the core maintainer for project-matching, which is being entirely reinvented in Ruby on Rails.

Our little project draws contributors from DSC UIUC and DSC NTU Singapore
